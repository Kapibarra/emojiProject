🇧🇾 Flag: Belarus

Copy


Interesting fact: This emoji has two conditions:

The first one is the official flag of the Republic of Belarus, displayed in the menu of the emoji section in the form of cloth with a red and green stripe, as well as an ornament.

The second one is the unofficial flag of the country (white-red-white), displayed when sending an emoji. When you click on the sent emoji, the flag flaps as if in the wind for a few seconds.


For the user to be able to share animated emojis, it is necessary to ensure that the Telegram app is updated to a 5.10 version or higher on both iOS and Android.

  1. If the version of the application on your device is the previous one, be sure to update the application to the current version;
  2. After installing the update, open any chat in the application and send one of the animated emojis there;
  3. If you already have the latest version of the application installed, but animated emojis are still not available for use, then go to the Settings section, select the Chat Settings item, and check the box next to the Large Emoji item.

Pay attention! If the user to whom you are sending a message with an animated emoji doesn’t have the latest version of the application, then alas, he will receive a regular static emoji.
